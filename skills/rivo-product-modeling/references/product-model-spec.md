# Product Model Specification

## Canonical Sections
- Attribute dictionary
- Component dictionary
- Compatibility map
- Derived values
- Version tags

## Minimum Attribute Fields
- `key`
- `type`
- `unit` (if numeric)
- `domain` (range or enum)
- `default`
- `uiHint`

## Required Version Tags
- `catalogVersion`
- `rulesVersion`
- `pricingVersion`
- `assetsVersion`
- `configurationVersion`

## Review Checklist
- Units validated
- Domains validated
- Enum IDs stable
- Backward compatibility assessed
- Example snapshot updated
