# 9. Roadmap до полной готовности (enterprise)

## MVP (v1)
- Product model + constraint engine
- reverse mode (3 решения)
- CPQ (price + BOM)
- PDF export (минимальная инструкция)
- saved/shareable configurations

## v2
- DXF export
- библиотека модулей (оборудование)
- compare configurations
- multi-language RU/EN
- админка каталога (версии)

## v3
- IFC export (BIM)
- OR-Tools optimization (min price/weight/max safety)
- ERP/CRM интеграции
- multi-tenant (дилерские прайсы)
- audit logs + governance
