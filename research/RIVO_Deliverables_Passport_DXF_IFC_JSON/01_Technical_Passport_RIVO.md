# Технический паспорт системы RIVO Fix / RIVO Set / RIVO Touch (на основе техкаталога 2025)

Источник: «Технический каталог. Решения для инженерной сантехники. 2025» (RIVO Systems).

## 1. Наименование и назначение

**RIVO Fix** — система быстрого монтажа конструкций из алюминиевого профиля и набора крепёжных элементов для создания оснований под прокладку инженерных коммуникаций (фальшстены, каркасы, колонны, перегородки, монтаж инсталляций).  
**RIVO Set** — инсталляционные модули (унитаз/унитаз AM/биде/писсуар) для скрытого монтажа.  
**RIVO Touch** — панели смыва для баков RIVO Set A.

## 2. Состав системы (номенклатура)

### 2.1 RIVO Fix — профиль и крепёжные элементы

- **100001.1** — Монтажный профиль (упаковка 1 шт)  
- **100098** — Монтажный профиль (упаковка 98 шт)  
  Назначение: сборка жёстких каркасов, фальшстен, оснований под обшивку ГКЛ.

- **100002** — Угловой элемент 180°/90°/45° (упаковка 100 шт)  
- **100002.1** — Угловой элемент 180°/90°/45° (упаковка 20 шт)  
  Назначение: соединение монтажного профиля под углами 45/90/180; литая конструкция для плотного прилегания; совместимость с профилями других производителей.

- **100013** — Заглушка (1 шт)  
  Назначение: закрытие торцов профиля; защита от влаги/пыли; закрытие острых кромок.

- **100014** — Торцовое соединение (1 шт)  
  Назначение: соосное соединение профилей; ускоряет сборку; снижает отходы при раскрое.

- **100004** — Крепление одинарное (1 шт)  
- **100004.1** — Крепление одинарное (10 шт)  
- **100004.2** — Крепление одинарное (100 шт)  
  Назначение: крепление профиля к стене/потолку; гальваническое защитное покрытие; комплектный крепёж.

- **100005** — Крепление двойное (1 шт)  
- **100005.1** — Крепление двойное (5 шт)  
  Назначение: крепление двух направляющих профилей к стене/потолку; гальваническое покрытие; комплектный крепёж.

- **100006** — Зажим (1 шт)  
- **100006.1** — Зажим (20 шт)  
- **100006.2** — Зажим (100 шт)  
  Назначение: сопряжение профиля с различными элементами крепёжных систем.

- **100008** — Монтажная пластина (1 шт)  
  Назначение: крепление водорозеток и встраиваемых частей смесителей; пластина из оцинкованной стали; элементы крепления с гальваническим покрытием.

- **100007** — Соединение универсальное (1 шт)  
- **100007.1** — Соединение универсальное (4 шт)  
- **100007.2** — Соединение универсальное (10 шт)  
  Назначение: соединение/усиление профилей для повышения общей жёсткости каркаса; гальваническое покрытие.

- **100009** — Телескопические опоры (1 шт)  
  Назначение: устанавливаются сверху в инсталляции RIVO Set для дополнительного усиления; регулировка по высоте; простой монтаж.

- **100010** — Комплект для крепления инсталляции к стене (1 шт)  
  Назначение: фиксация инсталляции RIVO Set к стене и/или к профилю RIVO Fix; комплектный крепёж; совместимость с профилями на рынке.

- **100011** — Опора для отдельной перегородки (1 шт)  
  Назначение: фиксация каркаса из профиля к опорной поверхности; критично для отдельно стоящих конструкций; совместимость с профилями на рынке.

- **100003** — Вставка резьбовая М8 (100 шт)  
- **100003.1** — Вставка резьбовая М8 (10 шт)  
  Назначение: применяется вместе с угловым соединением 100002; обеспечивает крепление хомутов труб, упрощая монтаж.

- **100012** — Комплект монтажных петель (1 шт)  
  Назначение: сборка нестандартных сантехнических люков разных размеров. Диапазоны: 180 ≤ A ≤ 600 мм; 200 ≤ B ≤ 600 мм.

### 2.2 RIVO Set — инсталляции

- **200001** — RIVO Set A инсталляция для унитаза со сливным баком  
  Особенности: бак 10 л; предустановка 6 л; настройки 4.5/7.5/9 л; малый слив 3 л; заливной клапан R 1/2" слева (возможность сверху); стальная рама до 400 кг; регулируемые опоры; «паспорт изделия» в комплекте.

- **200002** — RIVO Set AM (для медучреждений и МГН)  
  Как 200001 + кронштейны под поручни; комплект крепления к стене (8 шт).

- **203001** — RIVO Set B инсталляция для биде  
  Состав: рама; 2 опоры (регулировка 0–250 мм); шпильки M12; канализационный отвод DN50; регулируемые площадки под водорозетки.

- **201001** — RIVO Set D инсталляция для писсуара  
  Состав: рама; 2 опоры (0–250 мм); глубина установки 90 мм; шпильки M8; 2 регулируемые монтажные пластины; DN50; регулируемое по высоте соединение сифона.

### 2.3 RIVO Touch — панели смыва (для баков RIVO Set A)

- **RIVO Touch Lux**: 301001–301008 (варианты покрытий)  
- **RIVO Touch Line**: 302001–302007  
- **RIVO Touch Neo**: 303001–303007  
Габариты на чертеже: 150 × 220 × 6 мм (высота × ширина × толщина). Скрытое крепление. В комплект входят толкатели и крепёжные элементы.

## 3. Материалы и покрытия (по каталогу)

- Профиль и угловые элементы — алюминиевые сплавы.  
- Монтажная пластина — оцинкованная сталь.  
- Ряд крепёжных элементов — защитное гальваническое покрытие (коррозионная стойкость).

## 4. Требования к системе водоснабжения для эксплуатации RIVO Set

- Рекомендуется фильтр механической очистки на питающую магистраль с размером ячейки **≤ 100 мкм**.  
- Арматура рассчитана на давление **≤ 6 бар**.  
- Режим водоподготовки должен соответствовать **ГОСТ 2874-82**.

## 5. Гарантия и примечания

- Гарантия на инсталляции RIVO Set — **10 лет** со дня продажи (при наличии штампа торгующей организации в гарантийном талоне).  
- Производитель оставляет право изменения конструкции с целью улучшения потребительских свойств.

## 6. Инженерные правила применения (для проектирования и конфигуратора)

### 6.1 Принцип «пути нагрузки»
Любая нагрузка от оборудования/обшивки должна иметь непрерывный путь передачи:
оборудование → крепёж/узел → профиль → опора/анкер → основание.

### 6.2 Типовые назначения узлов
- 100002: углы/повороты/рамные контуры
- 100014: наращивание по оси (раскрой без отходов)
- 100007: усиление каркаса (инсталляция/перегородка/высокие стены)
- 100004/100005: крепление к стене/потолку
- 100011: базы для отдельно стоящих каркасов
- 100009: верхняя стабилизация/раскрепление инсталляций
- 100010: крепление инсталляции к стене/профилю
- 100008: платформа для водорозеток/смесителей
- 100003(М8): точки крепления хомутов/аксессуаров
- 100012: ревизионные люки
- 100013: завершение и безопасность торцов
