# RIVO FIX --- Инженерная модель для конфигуратора

## Полное инженерное описание системы

Версия: для передачи агенту-разработчику конфигуратора

------------------------------------------------------------------------

# 1. ОБЩАЯ ИНЖЕНЕРНАЯ КОНЦЕПЦИЯ

RIVO Fix --- модульная алюминиевая профильная система для создания
несущих конструкций инженерных коммуникаций.

Применение: - Инсталляции сантехники - Фальшстены - Отдельностоящие
перегородки - Колонны - Нестандартные контуры

Система работает как рамная конструкция: Профиль = несущая балка\
Соединение = узел передачи усилий\
Крепление = передача нагрузки в основание

------------------------------------------------------------------------

# 2. СОСТАВ СИСТЕМЫ

## 2.1 Несущий профиль

Артикул 100001 --- Монтажный профиль\
Материал: алюминиевый сплав\
Функции: - Вертикальная стойка - Горизонтальная перемычка - Рамный
элемент

Работает на изгиб и сжатие.

------------------------------------------------------------------------

## 2.2 Узлы соединения

100002 --- угловой 45°/90°/180°\
100014 --- торцовое соединение\
100007 --- универсальное усиленное

Узел должен передавать: - изгибающий момент - поперечную силу -
исключать люфт

------------------------------------------------------------------------

## 2.3 Опорные элементы

100004 --- крепление одинарное\
100005 --- крепление двойное\
100010 --- крепление инсталляции\
100011 --- опора перегородки\
100009 --- телескопическая опора

------------------------------------------------------------------------

# 3. РАСЧЕТНАЯ МОДЕЛЬ

## 3.1 Изгибающий момент

M = F \* L

## 3.2 Напряжение

σ = M / W

## 3.3 Прогиб

f = (F \* L\^3) / (3 \* E \* I)

Допустимый прогиб: L / 200

------------------------------------------------------------------------

# 4. ПАРАМЕТРИЧЕСКАЯ СТРУКТУРА

Wall: - width - height - depth - stud_step - load - support_type

------------------------------------------------------------------------

# 5. АЛГОРИТМ ГЕНЕРАЦИИ

1.  Ввод габаритов
2.  Расчет количества стоек
3.  Размещение вертикалей
4.  Добавление горизонталей
5.  Усиление зон нагрузки
6.  Проверка устойчивости

Количество стоек: N = floor(W / S) + 1

------------------------------------------------------------------------

# 6. УСТОЙЧИВОСТЬ

M_overturn = F \* h\
M_resist ≥ M_overturn

------------------------------------------------------------------------

# 7. ПРОВЕРКА ДИАГОНАЛЕЙ

d = sqrt(a² + b²)

# 10. ИНЖЕНЕРНЫЕ ПРАВИЛА

-   Нет висящих профилей
-   Минимум 2 точки фиксации
-   Проверка нагрузки 400 кг
-   Давление ≤ 6 бар
-   Наличие верхнего крепления

------------------------------------------------------------------------

# 11. ИТОГОВАЯ МОДЕЛЬ

RIVO Fix --- параметрическая рамная система с расчетной логикой,
CAD-совместимой геометрией и автоматической генерацией спецификации.
