# Stitch Prompt — Улучшение только Viewer (RU)

Сконцентрируйся на **центральном viewer** (2D/CAD) и связанной с ним панели инструментов. Не перерабатывай весь продукт.

## Цели
- Мгновенно понимать геометрию и габариты (front всегда основной).
- Переключение плоскостей XY/XZ/YZ, кнопки: Reset, Fit, Pan/Zoom, Measure.
- CAD‑режим: оси/размеры/слои, печатная читаемость, минимальная декоративность.
- Пересчёт: skeleton, “Пересчёт скелета…”, мягкие переходы.

## UI Viewer Toolbar (обязательное)
- View: Front / Side / Top (или XY/XZ/YZ)
- Toggle: Grid, Axes, Dimensions, Snapping, High Contrast
- Tool: Measure (точка‑точка), Ruler, Angle
- Export snapshot: PNG (для отчёта)
- Help: “Горячие клавиши”

## Взаимодействия
- Hover подсветка ребра/узла
- Drag handles (если включено редактирование)
- Tooltip с координатами/размером
- Мини‑легенда слоёв (в CAD‑режиме)

## Стиль
- Нейтральная инженерная палитра, тонкая линия 1px/1.5px.
- Тени минимальные; фокус на контрасте линий и подписей.
