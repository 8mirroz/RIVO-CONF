# AI‑Assisted Parametric Version (RU)
Цель: связать UI‑параметры ↔ solver/constraints ↔ геометрию ↔ BOM.

## 1) Параметры как “истина”
- Все параметры имеют: key, type, unit, min/max, step, default, dependencies.
- Любое изменение параметра запускает: validate → solve domains → update model → update BOM.

## 2) AI‑ассистент в UI
- Panel “Помощник”: 
  - объясняет, что сделать дальше (“следующий лучший шаг”)
  - предлагает значения (с учётом ограничений) — *только как предложение*
- Команды: “Сделай устойчиво”, “Уложись в 3м высоты”, “Снизь стоимость”.

## 3) Контракты API (минимум)
- POST /config/solve → возвращает domains + validity + messages + derived
- POST /config/export → возвращает ссылки на PDF/DXF/IFC + статус

## 4) UX‑ограничения ассистента
- Ассистент не меняет параметры без подтверждения пользователя.
- Любая рекомендация содержит: причину + эффект + риск.

## 5) Визуализация
- “Ghost preview” изменения: подсветка элементов, которые изменятся
- Diff‑panel: было/стало
