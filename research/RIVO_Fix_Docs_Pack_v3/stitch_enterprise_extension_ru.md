# Stitch Prompt — Enterprise‑версия (RU)

Сделай расширение до enterprise: роли, аудит, версии, согласования, интеграции.

## Роли и права
- Viewer (только просмотр)
- Engineer (редактирование, экспорт DXF/PDF)
- Manager (согласование, цена/скидки)
- Admin (каталог, правила, доступ)

## Обязательные экраны
- Организация/Проект/Команда
- История версий конфигурации (diff параметров + кто изменил)
- Approval flow: “На согласование” → “Одобрено/Отклонено” с комментарием
- Audit log (поиск/фильтры)
- Интеграции (ERP/CRM/PIM) — как настройки/статусы

## Фичи
- Multi‑tenant визуальная кастомизация (логотип, цвета, домен)
- SLA‑статусы системы + мониторинг (status pill)
- Экспорт‑пак: PDF+DXF+IFC одним пакетом, с checksum и версиями каталога/правил

## UI паттерны
- Табличные представления (DataGrid) с фильтрами/сохранёнными представлениями
- Комментарии и упоминания (@) внутри проекта
